## الوضع الحالي للمستودع

> يحتوي هذا المستودع حالياً على طبقة توثيقية وهندسية تأسيسية لضبط الجودة والهندسة المعمارية.
> تم تجهيز بروتوكول مراجعة شامل وأدوات تحليل قابلة للتنفيذ فور إضافة الكود التشغيلي للخدمات.

## دورة العمل الهندسية المعتمدة

1. **تثبيت خط أساس** (Baseline) مع تجزئة SHA-256 لكل ملف.
2. **تحليل ساكن منهجي** (ESLint / Pylint / Sonar-compatible outputs).
3. **استخراج خريطة الاعتماديات** (Dependency Graph JSON/SVG).
4. **إعادة هيكلة تدريجية موجهة بالاختبارات** مع الحفاظ على التوافق الرجعي.
5. **توثيق إلزامي** عبر CHANGELOG + ADR لكل قرار معماري.
